I developed and deployed smart contracts on the Stellar blockchain, ensuring secure, decentralized execution. The platform features an interactive map-based interface built with React and mapbox, allowing users to visualize and manage tasks efficiently. Additionally, I integrated real-time notifications and automated blockchain-based payouts, making the system highly interactive and self-sustaining.
